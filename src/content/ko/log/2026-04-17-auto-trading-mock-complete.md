날짜: 2026-04-11 ~ 2026-04-17

4월 11일부터 17일까지 7일. KIS OpenAPI 기반 자동매매 시스템을 모의투자로 돌렸다.

## 첫날 — 레짐 필터가 작동했다

첫날 KOSPI 하락장에서 레짐 필터가 작동했다. 매수 신호 없음. 현금 보유. 시스템이 아무것도 안 한 게 맞는 동작이었다.

## 첫 매수와 트레일링 스탑 첫 발동

카카오를 첫 매수했고, 오늘(04.17) 트레일링 스탑이 처음 발동했다.

- 고점: 50,900원
- 매도가: 49,850원 (자동)
- 수익률: +3.01%, 실현손익 +30,060원

재매수 금지 로직도 정상 작동했다. 오후에 셀트리온이 들어와 5슬롯이 다시 찼다.

## 버그 수정 4건

1주일 동안 버그 4개를 발견하고 당일 수정했다.

| 버그 | 수정 내용 |
|------|----------|
| 매수금액을 손실로 인식 | 매도 시에만 손익 누적하도록 수정 |
| 손실 한도 하드코딩 | config.MAX_DAILY_LOSS 참조로 통일 |
| 모의투자 예수금 고착 | 총평가 - 보유평가 계산 로직 추가 |
| 실계좌 계좌번호 형식 오류 | 숫자만 형식으로 수정 (대시 제거) |

## 검증 완료 항목

- 레짐 필터 — KOSPI MA20 기준 하락장 매수 차단 정상 작동
- 트레일링 스탑 — +4% 활성화 / 고점 대비 -2% 이탈 시 자동 매도 (카카오 발동 확인)
- 5슬롯 관리 — 매도 후 빈 슬롯 자동 재매수 정상
- 당일 재매수 금지 — 발동 후 쿨다운 정상
- 실계좌 API 연결 — 토큰 발급 / 잔고 조회 / TR_ID 분기 확인

## 미검증 항목
